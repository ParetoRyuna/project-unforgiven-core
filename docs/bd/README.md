# BD Toolkit (Pilot Outreach)

This folder contains copy-ready materials for founder-led BD and shadow-mode pilot outreach.

## Files

- `OUTREACH_TRACKER_TEMPLATE.csv`
  - Spreadsheet template for tracking outreach, replies, calls, and pilot progress.
- `SHADOW_MODE_MIN_INTEGRATION_EVENT_EXAMPLE.json`
  - Minimal server-side event payload example for shadow-mode integration (Option A).
- `SHADOW_MODE_MIN_INTEGRATION_SCORE_REQUEST_EXAMPLE.json`
  - API-assisted scoring request example for shadow mode (Option B).
- `PILOT_POST_EVENT_REPORT_TEMPLATE.md`
  - One-page post-event analysis/report template for design-partner pilots.

## Recommended Workflow

1. Copy `OUTREACH_TRACKER_TEMPLATE.csv` into Sheets/Notion/Airtable.
2. Start with shadow-mode only (`observe_only`).
3. Use JSON examples during technical alignment calls.
4. Deliver the post-event report within 24h after the pilot event.

## Pilot Positioning Reminder

- Assisted alpha pilots only (not public self-serve production onboarding yet)
- No custody of funds
- No private key sharing
- Removable / rollback-first integration
